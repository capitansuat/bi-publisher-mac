BI Publisher Template Builder

Privacy Policy  ·  Effective date: March 26, 2026

Summary No data collected

BI Publisher Template Builder is a Microsoft Word Add-in that runs entirely inside your local Word application. It does not collect, transmit, or store any personal information on external servers.

1. What data we collect

We collect no personal data. Specifically, the add-in does not:

  • Read, upload, or share the content of your Word documents
  • Collect names, email addresses, or any other personal identifiers
  • Send analytics, telemetry, or usage statistics to any server
  • Use cookies or tracking technologies

2. How the add-in works

All processing happens locally in your browser/Word WebView context:

  • XML data files you load are stored only in your browser's localStorage — they never leave your device.
  • Template fields and tags are written directly into your Word document using the Office.js API — no network call is made.
  • PDF preview is generated in-memory using Office.js and opened locally — the file is never uploaded.

3. Third-party services

The add-in's static files (HTML, JavaScript, icons) are hosted on GitHub Pages (pages.github.com). When Word loads the add-in, your browser makes a standard HTTPS request to GitHub's servers to fetch these files. GitHub Pages may log standard web server access logs (IP address, request time, user-agent) in accordance with GitHub's Privacy Statement. We do not have access to these logs.

4. Data retention

Since we collect no data, there is nothing to retain or delete. Any XML data you load is stored in your browser's localStorage and is cleared when you clear your browser data or remove the add-in.

5. Children's privacy

This add-in is a professional productivity tool. We do not knowingly collect any information from children under 13.

6. Changes to this policy

If this policy changes, the updated version will be published at this URL with a new effective date. We encourage you to review this page periodically.

7. Contact

If you have questions about this privacy policy, please open an issue on our GitHub repository.
